# CTRL+CURE — AI-powered health report analyzer built with Streamlit.

---

## The Problem

Lab reports are full of numbers most people don't understand.
CTRL+CURE reads your PDF, extracts the key health parameters,
and turns them into clear visuals and plain-English insights.

---

## Features

- PDF upload and automatic data extraction
- Health score visualization with gauge and charts
- Side-by-side report comparison
- Trend analysis with percentage change indicators
- Medication tracking with browser notification reminders

---

## Tech Stack

- Python
- Streamlit
- PyMuPDF
- Plotly
- Pandas

---

## Skills Demonstrated

- PDF Parsing and Text Extraction
- Data Processing and Analysis
- Interactive Data Visualization
- UI Development
- Analytical Thinking

---

## Getting Started
```bash
git clone https://github.com/saandralakhsshinia-ui/CTRL-CURE-Medical-doc-analyzer.git
pip install -r requirements.txt
streamlit run ctrl.py
```
